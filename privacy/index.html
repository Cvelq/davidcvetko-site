Privacy notice

What this site collects

Nothing, by default. This is a static site. It sets no cookies when you load a page, runs no tracking pixels, and serves every file (including fonts) from its own domain. The site is hosted on GitHub Pages, so GitHub serves the files like any host; beyond that, no third-party service is called when you read these pages.

The booking form

If you answer the two questions in the contact section and click "Book the call", you leave this site for my Calendly booking page. Your answers travel with the link so I have context for the call. Nothing from Calendly loads on this site and nothing is stored here. Calendly's own privacy policy governs the data you enter there.

External links

Links to outside tools (like the ROAS Reality Check) leave this site; those pages state their own practices.
