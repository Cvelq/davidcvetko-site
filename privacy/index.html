Privacy

Privacy notice

What this site collects

Very little. This is a static site. It sets no cookies when you load a page, runs no tracking pixels, and serves its own files (including fonts) from its own domain. The one measurement it uses is Cloudflare Web Analytics, which is cookieless: it counts page views in aggregate, stores nothing on your device, and does not follow you across sites. The site is hosted on GitHub Pages, so GitHub serves the files like any host.

The booking form

If you answer the two questions in the contact section and click "Book the call", you leave this site for my Calendly booking page. Your answers travel with the link so I have context for the call. Nothing from Calendly loads on this site and nothing is stored here. Calendly's own privacy policy governs the data you enter there.

External links

Links to outside tools (like the ROAS Reality Check) leave this site; those pages state their own practices.
